How to build MyAlumni bundled with Tomcat
=========================================================

- NOTE: before build MyAlumni, make sure set JAVA_HOME to JDK 1.5

- Extract a new tomcat 5.0.28, and change folder to, say, myAlumni-x.x-dev-jakarta-tomcat-5.0.28-yyyyddmm
    edit tomcat_home/conf/server.xml, then in tag Connector for port 8080, add this attribute:
     URIEncoding="UTF-8"
